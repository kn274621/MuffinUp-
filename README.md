# MuffinUp!
    This is the GitHub repository for the MuffinUp! project team to create a muffin mod for the game PlateUp!
    
    ## Deployment and Release
    
    The latest version of the MuffinUp! mod can be found on our releases page. We've put considerable effort into making the installation process as straightforward as possible for everyone.
    
    You can download the most recent release, [V1.0.0, directly from here](https://github.com/Ohio-University-CS/MuffinUp/releases/tag/V1.0.0).
    
    To get started with the mod, simply download the source code zip file from the release page, extract its contents, and then follow the installation instructions detailed within the PlateUp! modding community guides, as the process is standard for mods of this type.
